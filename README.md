# passcode_gen
Tool to generate random 4 digit passcodes.

# Install & Use
First clone the repo:
```
git clone https://github.com/AlexLandherr/passcode_gen.git
```
Change directory to repo:
```
cd passcode_gen/
```
Create a directory called `obj`:
```
mkdir obj
```
Then build & compile using `make`:
```
make
```

Run:
```
./passcode_gen
```
